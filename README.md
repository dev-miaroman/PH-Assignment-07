## 📖 Description

KeenKeeper is a modern Personal Relationship Management (PRM) web application that helps
users maintain meaningful friendships by tracking interactions, setting goals,
and visualizing communication patterns by reconnect bonding.
---

## 🛠️ Technologies Used

- **Next.js** — App Router, file-based routing
- **React** — Component-based UI with Context API for state management
- **DaisyUI CSS**— Utility-first styling
- **Tailwind CSS** — Utility-first styling
- **React-Spinners**-Interactive loading for analytics
- **Recharts** — Interactive pie chart for analytics
- **React Toastify** — Toast notifications
- **Vercel** — Deployment with git live domain controll

---

## ✨ Key Features


### 1. 📊 Friendship Analytics
Visual pie chart showing the breakdown of your interactions by type (**Call**, **Text**, **Video**). Get a clear picture of how they're staying in touch with the people in their life.

### 2. 👥 Friend Tracker
Keep a personal list of friends with details like tags, relationship goals, and how many days since they last contact. Friends are marked as **On Track**, **Almost Due**, or **Overdue** so they always know who needs attention.

### 3. 📅 Interaction Timeline
Log every check-in — whether it's a **Call**, **Text**, or **Video** — and view a full history of their interactions. Filter the timeline by interaction type to quickly find what they're looking for.

## 📦 Installation & Setup
Open [http://localhost:3000](http://localhost:3000) in your browser.

```bash
# Clone the repository
git clone https://github.com/dev-miaroman/PH-Assignment-07.git

# Navigate to project
cd keen-keeper

# Install dependencies
npm install

# Run locally
npm run dev

#vercel deploy
npm install --g vercel
vercel login
vercel --prod
```
